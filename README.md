# PSA-Can2004-AEE2004-Cluster-for-simracing-via-Simhub---Peugeot-407-Simracing-cluster-prototype
A prototype version using an AEE/CAN2004 Peugeot 407 cluster to be used for simracing with simhub. Every step is clearly explained, and current functionality includes working gauge needles(rpm,speed,fuel,temps) and blinkers, with more to be added. Hardware is done by myself, the code is generated by Claude with help from PROTOTUX and Ludwig V repos
The PDF contains all the informations needed about the project
